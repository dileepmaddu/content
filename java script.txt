 variables:
**************
   variables are used to store data
     ex. string
         number
         boolean
         arrays
         objects
                   --- -
 we will declare variables in 3 ways 
                   1.var
                   2.let
                   3.const
                   4. <no keyword>

let and const are  released in ES6 version
syntax:
        var/let/const variable_name = value;
        ex: 
           var x = "careerIT";
           document.write(x); //careerIT

rules of identifier
 1.identifier must have a-z,A-Z.0-9,$and_
 2.identifier should not start with digits
                  ex.
                    var 123abc; //wrong
                    var abc123
                    var_123
                    var_123$abc
                    var _123#abc; //wrong
DATATYPE:
***********
datatype representing type of data
we have two types of datatype
     1.primitivedatatypes
     2.non-primitivedatatypes

 primitive data types:
***********************
     primitive data types representing plain data/ normal data
     primitive datatype won't supports any predefined properties/predefined functions
     primitive dataTypes are divided into
                    1.string
                    2.number
                    3.boolean
                    4.undefined
                    5.null
                    6.bigint
                    7.symbol
non primitive data types:
*************************
                  
    non primitive datatypes are the objects
    non primitive datatypes supports predefined properties and predefined functions
                    ex.
                    arrays
                    objects
                    functions
                    classes
                    interfaces
